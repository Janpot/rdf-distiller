Overview

Spira::Errors represents a collection of validation errors for a Spira resource. It tracks a list of errors for each field.

This class does not perform validations. It only tracks the results of them.

def add(property, problem)

def each
  @errors.map do |property, problems|
    problems.map do |problem|
      property.to_s + " " + problem  
    end
  end.flatten
end
