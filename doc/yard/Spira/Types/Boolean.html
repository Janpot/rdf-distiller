Overview

A Spira::Type for Boolean values. Values will be expressed as booleans and packaged as XSD.boolean RDF::Literals.

A Resource property can reference this type as Spira::Types::Boolean, Boolean, or XSD.boolean.

def self.serialize(value)
  if value
    RDF::Literal.new(true, :datatype => XSD.boolean)
  else 
    RDF::Literal.new(false, :datatype => XSD.boolean)
  end
